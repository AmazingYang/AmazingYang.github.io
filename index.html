Github——希望是我最后的网络家园

发表于 2016-03-16   |   分类于 生活感悟   |     |   阅读次数

从我98年接触互联网开始，同每一个网民一样，也希望能够在网上安一个家，记录自己的生活感悟，分享学习心得，通过网络展示自己、认识朋友。但是运维一个网站显然不是那么容易，当时还是高中生的我只能用得起免费的服务器空间。这些免费的空间总是做不了多久就消失在互联网的浪潮里，于是我只能将我的网站搬来搬去。

后来随着博客的流行，各种托管类的网站也雨后春笋一般地冒出来，与其自己运维一个网站，当然不如托管来的方便，以下是我用过的：

  1. QQ空间：从中国有了互联网开始，人人都有QQ号，很多人也就有了QQ空间。很难说QQ空间到底是个什么东西。在我看来，就是一个大杂烩，有小年轻写着爱来爱去的文字，也有做广告的、写鸡汤文的，但它有个最大的缺点：用现在的话说就是Low，只适合年轻的小朋友玩玩，缺少高大上的元素。我上了大学基本上就放弃它了。
  2. 微软的Live Spaces：曾经我以为找到了永远的网络家园，谁TMD知道微软的空间也能黄啊，只能说信息技术发展的太快了，在这个领域，谁都当不了永远的老大。微软在互联网领域缺少一个能够掌握互联网发展方向的领袖。
  3. 人人网、开心网之类的SNS：人人网曾经做的非常好，可是山寨的东西总是不长久，说到底只是疲于模仿，缺少创新的思考。这也是为什么Facebook依然如日中天，但人人网已经淡出视野了。而且这类SNS网站倾向社交，每天上去一看都是各种分享和朋友的心情状态，很像现在的微信朋友圈。可我想看的不仅仅是吐槽秀恩爱装逼，我更想看一些有营养的文章，但是几乎找不到。而且还有一个最大的问题，像我这样的技术宅要是发表一篇技术心得，我的那些非技术的朋友会不会直接吓跑，或者留言“你好像技术很厉害啊，会给MAC装Windows系统吗？”(崩溃表情)
  4. CSDN、博客园：这类网站倒是适合发表技术文章，可是生活中需要交流和分享的不仅仅是技术。比如某段时间我对葡萄酒很感兴趣，我在这类网站发表一篇如何挑选葡萄酒的文章显然不合适。

说到底，还是个人网站适合个人，这句话是不是有点怪怪的？可是自己建站就得跟服务器供应商打交道定期续费，同时还得学习网站部署发布等技术。拜托，我只是想安静的写点文字。在人生的某一阶段，记录一些心得感悟，有人看最好没人看也无所谓。所幸，Github出现了！目前看来最好的选择就是用Github自己建站了。Github是个伟大的网站，代表着互联网的分享精神，希望这里是我最后的网络家园。我的这个小站是用Hexo框架搭建的，采用了NexT主题。感谢Github、感谢Hexo和NexT的作者，你们让搭建个人网站变得如此轻松！

另外，知乎上有人曾问过在Github上建站是不是有道德方面的问题，有个答案我非常赞同，我直接引用代表我的立场了。

单论github上写博客，完全符合github的宗旨，哪怕不分享代码，而仅仅是写博客。但作者至少应该对得起读者，不要把博客搞成自己的垃圾堆就好。
那些指责别人不该在github上挂博客的人其实是有另一种担忧：github不处于工信部监管之下，加之其近来巨火，其中的任何内容都有可能星火燎原。在奴隶看来，任何自由都是不道德的。故此那些道德与否的论辩只是在强调要守规矩，虽然这规矩的正当性值得质疑。
同时还有一些技术工作者，素来奉行不谈国事。面对sourceforge被墙，Golang官网被墙，我们也只能苦笑。国人素来对免费的东西趋之若鹜，若要把github这样一个免费开放没有审查限制的网站推广给大家挂博客，势必会迅速充斥大量政治敏感和少儿不宜内容。倘若如此，github被墙的命运就成了必然。为了逃避这一厄运，才出现了挂博客不道德的论调吧。
